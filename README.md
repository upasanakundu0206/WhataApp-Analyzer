\# 💬 WhatsApp Chat Analyzer



An ML-powered WhatsApp chat analysis web app built with Python and Streamlit.



\## Features

\- 📊 Message \& word statistics

\- 📅 Monthly \& daily timelines

\- 👥 Most active users analysis

\- ☁️ Word cloud visualization

\- 😊 Emoji frequency analysis

\- 🧠 Sentiment analysis (VADER)

\- 🤖 ML Message Predictor (Naive Bayes)

\- 🔗 Link \& domain analysis



\## How to run

pip install -r requirements.txt

streamlit run app.py



\## Built With

Python · Streamlit · scikit-learn · pandas · VADER · WordCloud · Plotly
